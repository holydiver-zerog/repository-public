●特典ファイルについて
特典ファイルとして『Python2年生 デスクトップアプリ開発のしくみ 体験してわかる！会話でまなべる！』の第2章　06「配色を選べるよ」でも触れた配色について、いろいろなテーマの配色を試せるアプリを用意しました。
リストからテーマを選択すると、その配色でダイアログが表示されます。
気に入ったら、入力欄のテーマ名を選択してコピーし、お使いください。「OK」か「Cancel」でダイアログが閉じます。

●特典ファイルのテスト環境
　特典ファイルは以下の環境で、問題なく動作することを確認しています。

・OS：Windows 11/10、macOS Monterey（12.2.x） 
・Python：3.11.0/3.10.4
・各種ライブラリとバージョン
  pysimplegui：4.60.3
